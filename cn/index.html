关于我们

一个未来的统一团队是一个专注于技术的小型独立团队，由少数成员组成。 我们主要专注于网站和软件技术。

一个未来的统一团队的主要任务是将用户的隐私权还给我们的用户。 我们重视每个人的隐私。 我们认为，无论是在现实世界还是在互联网上，每个人都应该将自己的隐私权重新掌握在自己手中，因此我们希望实现这一点。


开源

一个未来的统一团队项目是完全开源的。 我们所有的代码都是公开的，供所有人使用、阅读和学习。 我们目前不打算有任何闭源项目。 我们所有的源代码都可以在我们的 Github 组织中获得。 我们还相信，公开提供我们的源代码也会在我们的用户和团队之间创建一个透明层。

由于我们所有的项目都是开源的，我们还优先考虑我们所有项目的隐私。 我们在我们的网站上收集的用户数据少于任何其他网站。 我们也没有共享或出售在我们网站上收集的任何数据，并且我们将所有数据安全地保存在我们的数据库中。 我们不会为每个用户创建一个特殊的虚拟个人资料。


免费

一个未来的统一团队制作和提供的所有项目和服务完全免费，没有任何隐藏成本。 我们所有的服务和项目对所有人免费，包括企业家、企业和穷人。 无论如何，每个人都应该使用我们所有的隐私友好型项目和服务。 我们的服务和网站不包含任何付费专区或溢价。


安全环境

我们希望在互联网上提供第一个最安全的环境。一个未来的统一团队严格控制我们网站上的所有内容。 我们还制定了严格的社区准则，以确保我们的社区安全和清洁。 我们还对通过我们网站的每个内容进行了审核。


100% 正常运行时间

在需要使用服务时讨厌停机？ 我们在这里为您提供保障。 目前，我们所有的项目和服务每天都有 100% 的正常运行时间。 我们还维护和跟踪以使我们的网站在全年每天都保持正常运行。 当我们的网站出现故障时，我们也会尽快通知大家。
